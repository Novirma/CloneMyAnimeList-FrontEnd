-Tutorial Install ReactJS :

* Untuk membuat ReactJS maka gunakan Building Tools ViteJS
* Install dengan "pnpm create vite"
* Buat nama Project Folder
* Pilih React
* Pilih Javascript
* Lakukan "pnpm install" untuk menginstall Node Modules
* Running aplikasi dengan perintah "pnpm run dev"
* Bukak main.jsx lalu hapus index.css
* Bukak app.jsx lalu hapus import logo dan app.css
* Pada app.jsx hapus semua isi Function app nya
* Hapus app.css dan isi assets nya
* Gambar di public juga di hapus


-Apabila Aplikasi Telah Selesai :

* Build aplikasi agar bisa di deploy ke server dengan query "pnpm run build"
* Agar terbentuk folder "dist" yang sudah merangkap semua aplikasi
